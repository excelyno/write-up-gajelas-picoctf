output awal menujukkan error 
termasuk
chunk name tidak valid
crc error
dan invalid chunk length

ini mengarah pada kesumpulan bahwa
file ini adalah file png yang sengaja dikorupsi pada level struktur chunk

berikut adalah informasi terkait struktur dasar png 
1. png signature (8 bytes)
2. chunk berurutan 
	- IHDR
	- sRGB / gAMA / pHYs ( opsional )
	- IDAT (data gambar)
	- IEND

setiap chunk memiliki format
[4 byte LENGTH]
[4 byte TYPE]
[n byte DATA]
[4 byte CRC]

jika salah satu bagian rusak, maka tidak bisa dibacalah png itu

kita analisis dengan xxd ikan.png | head

dan kita menemukan ada beberapa kejanggalan yaitu header tidak sesuai , 
chunkname tidak sesuai standar png 
nilai lenght tidak masuk akan 
dan crc tidak cocok
