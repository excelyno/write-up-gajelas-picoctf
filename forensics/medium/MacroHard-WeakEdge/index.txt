dan saya mencari tau terkait model file extensi ini dan saya menemukan bahwa
file pptm ini pada dahulunya adalah file zip. 

Walaupun file tidak menulis "zip archive", semua PowerPoint 2007+ adalah ZIP.

nah ini adalah indikasi besar kita dapat mengubah tipe filenya. 
kita ubah menjadi zip dengan mv lalu kita ekstrak
